## Software

### Audio analysis

- [Praat](https://www.praat.org/): standard open source tool for phonetic analysis of speech

### Behavioral experimental platforms

- [jsPsych](https://www.jspsych.org/latest/): jsPsych is a JavaScript framework for creating behavioral experiments that run in a web browser
- [OpenSesame](https://osdoc.cogsci.nl/): OpenSesame is a program to create experiments for psychology, neuroscience, and experimental economics
- [oTree](https://www.otree.org/): oTree is a behavioral research platform that lets you build any kind of task or interaction, even if it's something nobody else has made before
- [PsychoPy](https://psychopy.org/index.html): Open-source package for running experiments in Python (a real and free alternative to Matlab) that combines the graphical strengths of OpenGL with simple Python syntax to give scientists a free and simple stimulus presentation and control package. It is used by many labs worldwide for psychophysics, cognitive neuroscience and experimental psychology

### Data wrangling, modeling, exploration and analysis

- [Heurist](https://heuristnetwork.org/): free, open-source database platform for building, managing and publishing richly-structured humanities research data
- [nodegoat](https://nodegoat.net/): web-based research environment for the humanities
- [OpenRefine](https://openrefine.org/): powerful free, open source tool for working with messy data
- [SnakeMake](https://snakemake.github.io/): workflow management system to create reproducible and scalable data analyses

### Digital collections, exhibitions and image management

- [Omeka](https://omeka.org/): free, open source web-publishing platform for library, museum, archive and scholarly collections and exhibitions
- [Tropy](https://tropy.org/): free, open source research photo management for archival images

### Electronic lab notebooks

- [Chemotion ELN](https://chemotion.net/#eln): Electronic Lab Notebook for chemists funded by DFG and NFDI4Chem
- [elabFTW](https://www.elabftw.net/): free and open source electronic lab notebook
- [sampleDB](https://scientific-it-systems.iffgit.fz-juelich.de/SampleDB/): web-based electronic lab notebook with a focus on sample and measurement metadata by Forschungszentrum Jülich

### Geospatial research

- [Allmaps](https://allmaps.org/): open source tools for curating, georeferencing and exploring IIIF maps
- [QGIS](https://qgis.org/): free and open source geographic information system

### Image and video annotation and analysis

- [CVAT](https://github.com/cvat-ai/cvat): MIT-licensed, self-hosted web tool for image and video annotation (object detection, segmentation, tracking)
- [LabelMe](http://labelme.csail.mit.edu/): open-source web tool by MIT CSAIL for annotating images with polygons, bounding boxes and segmentation masks
- [VGG Image Annotator (VIA)](https://www.robots.ox.ac.uk/~vgg/software/via/): lightweight, dependency-free, BSD-licensed tool for manually annotating image and video data
- [Vian](https://github.com/Movie-Analytics/VIAN): A video annotation software built with Electron and Vue.js
- [ELAN](https://archive.mpi.nl/tla/elan): annotation tool for audio and video recordings
- [TIB AV-Analytics (TIB-AV-A)](https://service.tib.eu/tibava): video analysis platform for systematic film and video analysis

### Linguistic corpora

- [AntConc](https://www.laurenceanthony.net/software/antconc/): freeware, multiplatform tool for corpus linguistics research and data-driven learning
- [EXMARaLDA](https://exmaralda.org/en/about-exmaralda/): EXMARaLDA is a system for working with oral corpora on a computer
- [IMS Open Corpus Workbench (CWB)](https://cwb.sourceforge.io/): open-source tools for managing and querying large text corpora with linguistic annotations
- [NoSketch Engine](https://nlp.fi.muni.cz/trac/noske): open-source version of the commercial Sketch Engine corpus query system

### Network analysis and visualization

- [Gephi](https://gephi.org/): open source software for visualizing and analyzing large network graphs
- [Cytoscape](https://cytoscape.org/): open source platform for complex network analysis and visualization, originating in biological research

### (Online) Surveys

- [formr](https://formr.org/): chain simple surveys into long runs, use the power of R to generate pretty feedback and complex designs

### Qualitative data analysis

- [QualCoder](https://github.com/ccbogel/QualCoder): free, open source qualitative data analysis software for text, images, audio and video
- [Taguette](https://www.taguette.org/): free and open source text tagging tool for qualitative research

### Statistical analysis

- [JASP](https://jasp-stats.org/): open-source statistics software for classical and Bayesian analyses with an intuitive GUI
- [jamovi](https://www.jamovi.org/): open statistical software built on R, a free alternative to SPSS/SAS
- [PSPP](https://www.gnu.org/software/pspp/): free GNU tool for statistical analysis of sampled data, SPSS alternative

### Systematic review and bibliometrics

- [ASReview](https://asreview.nl/): open source AI-assisted screening tool for systematic reviews using active learning
- [VOSviewer](https://www.vosviewer.com/): free software for constructing and visualizing bibliometric networks
- [bibliometrix](https://www.bibliometrix.org/): open-source R tool for quantitative research in scientometrics and bibliometrics

### Text annotation, analysis and publication

- [CATMA](https://catma.de/): web-based, open source tool for undogmatic text annotation and analysis (University of Hamburg)
- [ediarum](https://www.ediarum.org/index.html): digital work environment consisting of several software components that allows scholars to edit transcriptions of manuscripts and printed works in TEI-compliant XML, add textual and critical apparatus as well as indexes, and publish them online and in print
- [INCEpTION](https://inception-project.github.io/): A semantic annotation platform offering intelligent assistance and knowledge management
- [LEAF Writer](https://dev.leafwriter.users.h-da.cloud/): XML & RDF online editor of the Linked Editing Academic Framework
- [MALLET](https://mimno.github.io/Mallet/): Java-based package for statistical NLP, document classification, clustering, topic modeling and information extraction
- [Recogito](https://recogito.pelagios.org/): open source tool for collaborative semantic annotation of texts and images, incl. geo-annotation via gazetteers (Pelagios Network)
- [TEI Garage](https://teigarage.tei-c.org/): Web tool for converting to and from TEI XML
- [TEI Publisher](https://teipublisher.com/index.html): TEI Publisher is designed as a working environment for creating and publishing of digital scholarly editions
- [TextGrid](https://textgrid.de/): open source virtual research environment for digital scholarly editing
- [Voyant Tools](https://voyant-tools.org/): web-based reading and analysis environment for digital texts
- [Hypothes.is](https://web.hypothes.is/): open-source browser-based tool for annotating webpages, PDFs and EPUBs
- [Annotation Studio](https://www.annotationstudio.org/): open-source collaborative web annotation suite for close reading and textual interpretation (MIT Hyperstudio)
- [Scalar](https://scalar.me/): free, open source authoring and publishing platform for long-form, born-digital scholarship

### Text recognition (OCR, HTR etc.)

- [Calamari](https://github.com/Calamari-OCR/calamari): line-based OCR/HTR engine with deep-learning models, used e.g. in OCR4all
- [dinglehopper](https://github.com/qurator-spk/dinglehopper): OCR evaluation tool
- [eScriptorium](https://gitlab.com/scripta/escriptorium): A project providing digital recognition of handwritten documents using machine learning techniques.
- [kraken](https://kraken.re/): trainable OCR/HTR engine optimized for historical documents and non-Latin scripts; powers eScriptorium
- [OCR4all](https://www.ocr4all.org/): open source tool providing a (semi-)automatic OCR workflow for historical printings
- [OCR-D](https://ocr-d.de/en/): coordinated framework and toolset for the mass digitization and OCR of historical documents
- [Tesseract](https://github.com/tesseract-ocr/tesseract): widely used open source OCR engine
- [Transkribus](https://www.transkribus.org/): Platform for written heritage
